# CampusLifeAgent

> 基于 Spring Boot + AI Agent 的校园学习与生活一站式智能助手平台

## 📖 项目简介

面向高校学生的智能化一站式平台，通过 AI Agent 技术整合校内日常学习与生活信息，解决信息碎片化问题，提供个性化学习答疑、生活助手与课程项目辅助能力。  


项目包含三大核心能力：

### 1. 校园知识库 RAG 智能问答
将课程信息、课件、常见 QA 等非结构化文档向量化存储与检索，构建校园知识库 Agent，实现精准的文档问答与信息查询加速。

### 2. 多场景对话 Agent（学业咨询 / 生活助手）
基于 ReAct 推理范式实现对话 Agent，支持场景无缝切换与多轮对话；通过 MCP 协议接入课程数据库、空间预约系统、校园活动日历，实现多数据源查询与工具调用。

### 3. 课程项目规划 Agent（Plan-Execute-Replan）
面向“模糊目标 → 可执行计划”任务，采用 Plan-Execute-Replan 推理范式，将课程项目目标拆解为「文献调研-技术选型-项目规划」步骤，并调用知识库与工具链生成带资源清单的参考方案。

## 🚀 核心特性

- ✅ **RAG 校园问答**：向量检索 + 多轮对话 + 精准文档引用
- ✅ **多场景对话**：ReAct 推理 + 场景切换 + 上下文维护
- ✅ **工具集成（MCP）**：课程数据、空间预约、校园日历等多数据源查询
- ✅ **项目规划**：目标拆解 + 自动执行 + 动态重规划（Plan-Execute-Replan）
- ✅ **统一 API**：提供 RESTful API（可扩展 Web 测试界面）
